Tilley Renamer Readme V.4:

Steps for renaming and preparing pdf files for Enterprise and Livecount:

1) Create a new folder somewhere

2) Extract named pdf files to the folder created in step 1

3) Copy the "T-renamer.exe" files into the folder created in step 1

4) Run the "T-renamer.exe" file in folder created in step 1

5) Follow the instructios
